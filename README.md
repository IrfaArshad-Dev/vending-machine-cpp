# Vending Machine — C++

A console-based vending machine simulation built in C++ as part of my SE coursework at Riphah International University.

## Features
- Browse available items and prices
- Select item and enter payment
- Returns change automatically
- Handles invalid inputs

## How to run
1. Download the .cpp file
2. Open in any C++ compiler (VS Code, CodeBlocks, Dev C++)
3. Compile and run

## Tech used
C++ | OOP concepts
